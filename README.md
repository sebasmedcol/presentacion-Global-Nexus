# MVM NEXUS AI - Presentación (GitHub Ready)

Presentación web (HTML/CSS/JS) que mezcla la versión de presentación + demo interactiva. Estética inspirada en Material Design y lista para publicar en GitHub Pages.

## Archivos incluidos

- `index.html` — Página principal (presentación + demo)
- `style.css` — Estilos (Material-like)
- `script.js` — Lógica de la presentación y demo interactiva
- `README.md` — Este fichero

## Cómo desplegar en GitHub Pages

1. Crear un repositorio nuevo (por ejemplo `mvm-nexus-ai-presentation`).
2. Subir todos los archivos a la rama `main` (o `gh-pages` si prefieres).
3. En GitHub: **Settings → Pages**, seleccionar la rama `main` + carpeta `/ (root)` y guardar.
4. Esperar unos minutos y tu sitio estará disponible en `https://<tu-usuario>.github.io/<tu-repo>/`

## Notas funcionales

- Demo de proyectos es solo simulada (no hay backend). Puedes conectar fácilmente a una API REST later.
- Reglas incorporadas: solo PDF/XLSX como evidencia (nota), cancelación requiere justificación, no cancelar en "En Revisión".
- Para producción: agregar autenticación, validaciones de archivos, almacenamiento y MLOps para guardado de métricas y auditoría.
